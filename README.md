# Hotel Booking Frontend

This folder is ready to open and run directly in VS Code.

## Run in VS Code

1. Open this folder in VS Code.
2. Open the terminal in VS Code.
3. Run `npm install` if `node_modules` is missing.
4. Run `npm run dev`.
5. Open the local Vite URL shown in the terminal.

## Build

Run:

```bash
npm run build
```

## Project notes

- Local hotel images are stored in `public/images`.
- Main React app files are in `src`.
- This is a frontend-only demo with mock hotel and booking data.
